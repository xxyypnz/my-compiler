## 负向测试

命令：

```bash
make test-errors
```

结果：通过。

覆盖：

- 重复声明
- 未声明变量
- 赋值类型不匹配
- `read` 非 int 变量
- `add` 非 set 变量
- `set != set`
- 非变量集合相等
- 除 0

## 完整测试

命令：

```bash
make test-all
```

结果：通过。



## 验收命令

进入程序目录运行：

```bash
cd submission/程序
make clean && make
make test
make test-errors
make test-all
```

单步模式：

```bash
./l26c -s ../测试/正向测试/test1.l26
```

只打印 P-Code：

```bash
./l26c -p ../测试/正向测试/test2.l26
```
